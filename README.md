# AdvancedPNGExport
A Paint.NET plugin that allows exporting RGB, RGBA, and Grayscale 8 bit textures compatible with BeamNG Texture Cooker.
Master branch is compatible with Paint.NET 5.1.11 and x86_64 and ARM64 CPUs.
0.4.21_legacy branch is tested with Paint.NET 4.0.21.
